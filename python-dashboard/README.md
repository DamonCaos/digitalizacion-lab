# Python Dashboard · Base

Este proyecto es una base para crear dashboards industriales.

## Requisitos recomendados

```bash
pip install pandas plotly dash
```

## Uso básico

```bash
python app.py
```

- Modifica `load_sample_data()` para conectarte a SQL o a ficheros reales.
- Usa Dash o Streamlit para montar un dashboard completo.
