# Servidor OPC UA simulado

Base para un servidor OPC UA de pruebas.

```bash
pip install opcua
python simulator.py
```

- Expone algunas variables típicas de una línea de extrusión:
  - TempCabezal
  - VelocidadLinea
  - ScrapPorcentaje
- Conéctate desde un cliente OPC UA para probar tus aplicaciones .NET/Python.
